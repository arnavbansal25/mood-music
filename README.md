# Mood Music <img src="https://github.com/arnavbansal25/mood-music/blob/main/logo.png" width=50>
Mood-Music is a React based web application with capabilities to analyze and detect emotion from live webcam feed in real time. The application takes emotion and weather captured at userâ€™s location and concatenate them into a string. That combined string is then passed into an API to discover the relevant songs which the user might like. The site provides users a friendly interface with engaging visuals and easy navigation.
